## Descrição
Projeto em Assembly MIPS implementado como calculadora programador didática, exibindo passo a passo das operações.

---

## Funcionalidades

### Q1 – Conversões de Base  
- Decimal → Binário  
- Decimal → Octal  
- Decimal → Hexadecimal  
- Decimal → BCD  

### Q2 – Complemento de 2 (16 bits)  
- Converte número decimal para representação com sinal  
- Para números negativos: inverte bits, soma 1 e exibe resultado final

### Q3 – IEEE754 Float e Double  
- Exibe o valor em float e double  
- Mostra bits de sinal, expoente, expoente sem viés e fração  
- Imprime os 32 bits (float) e 64 bits (double)
